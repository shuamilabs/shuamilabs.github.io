Jana Fail CSR di Pelayan Windows

posted on 30 May 2020 by Ahmad Shuami

Assalamualaikum dan Salam Sejahtera.

Perkongsian hari ini adalah berkenaan langkah-langkah untuk menghasilkan fail CSR di dalam pelayan Windows. Fail CSR ini diperlukan semasa anda ingin memohon atau membeli sijil SSL (secure socket layer) untuk menggunakan https pada domain anda. Anda boleh juga menggunakan tool yang sedia ada di internet. Jom kita lihat cara-cara menggunakan perintah di command prompt. Caranya mudah dan cepat. sila ikuti langkah-langkah di bawah.

Langkah #1

Pastikan server anda telah dipasang dengan Apache dan Openssl.

Langkah #2

Buka command prompt anda (as administrator). Setkan path bagi OPENSSL_CONF seperti di bawah.

Path ini adalah bergantung kepada di mana anda memasang Apache di komputer anda.

Langkah #3

Buka direktori bin di mana pelayan web anda berada. Sebagai contoh, saya telah memasang pelayan web Apache di direktori C:\Apache24. Jadi, saya buka direktori bin seperti di bawah.

Langkah #4

Seterusnya, taipkan perintah seperti di bawah untuk menghasilkan fail .csr dan fail .key. Fail-fail ini perlu disertakan semasa permohonan atau pembelian sijil SSL nanti.

Anda akan ditanya dengan beberapa soalan sebelum fail-fail dihasilkan. Jawab soalan-soalan yang telah diberikan. Antara soalan yang perlu dijawab, sebagai contohnya;

Langkah #5

Tekan [Enter] untuk menghasilkan fail. Jika anda menggunakan perintah seperti di Langkah #4, dua (2) fail iaitu myserver.csr dan myserver.key akan dijana di dalam direktori C:\Apache24\bin. Sila simpan fail-fail ini sebaik-baiknya.

Ok, anda sudah berjaya menghasilan fail csr bagi tujuan memohon atau pembelian sijil SSL. Selamat mencuba.

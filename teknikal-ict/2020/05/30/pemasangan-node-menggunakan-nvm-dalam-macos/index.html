Pemasangan Node Menggunakan NVM dalam macOS

posted on 30 May 2020 by Ahmad Shuami

Assalamualaikum dan Salam Sejahtera.

Hai semua. Hari ini saya ingin kongsikan bagaimana anda boleh memasang node di dalam macOS. Banyak kaedah yang boleh digunakan, antaranya menggunakan homebrew, installer node dan juga menerusi nvm. Perkongsian saya kali ini adalah cara memasang node menggunakan nvm.

Langkah #1

Muat turun dan pasang skrip nvm menggunakan curl. Buka terminal anda dan jalankan perintah di bawah. Tunggu sehingga skrip dipasang selesai.

  • v0.35.2 adalah versi terkini semasa hantaran ini ditulis. Anda boleh mengemaskini versi selepas ini.

Langkah #2

Periksa versi nvm anda di terminal menggunakan perintah seperti di bawah.

Langkah #3.1

Seterusnya, kita akan memasang node.js. Jika anda ingin memasang node yang terbaharu, gunakan perintah di bawah.

Langkah #3.2

Jika anda ingin memasang versi stabil yang terbaharu, gunakan perintah di bawah.

Langkah #4

Anda boleh mengesahkan versi node dan npm (npm dipasang secara automatik semasa pemasangan node) dengan perintah seperti di bawah.

** Nota yang berguna

Jika anda ingin mengemaskini nvm kepada versi terbaharu, sila ikuti langkah di Langkah #1. Anda boleh semak versi terbaharu nvm di sini. Jika anda ingin mengemaskini node kepada versi terbaharu, sila ikuti langkah-langkah di bawah.

Jika anda memasang node menggunakan kaedah seperti di Langkah #3.1, kemaskini node dengan perintah.

Jika anda memasang node menggunakan kaedah seperti di Langkah #3.2, kemaskini node dengan perintah.

Untuk melihat senarai semua node yang telah dipasang.

Jika anda telah memasang banyak versi node, anda boleh setkan default node yang diinginkan. Jalankan perintah di bawah.

  • set default node kepada node terbaharu yang telah dipasang
  • set default node kepada node versi stabil terbaharu yang telah dipasang

Jadi, itu sahaja secara ringkas bagaimana anda boleh menggunakan nvm untuk memasang node. Cara ini membolehkan kita memasang banyak versi node di dalam komputer kita. Selamat mencuba.

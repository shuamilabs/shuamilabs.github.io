Pemasangan Angular CLI

posted on 30 May 2020 by Ahmad Shuami

Assalamualaikum dan Salam Sejahtera.

Hai, anda semua sihat hari ini? Anda ingin berjinak-jinak dengan Angular? Bagi anda yang ingin mencuba rangka kerja Angular ini, perkara pertama adalah, anda perlu memasang dahulu angular cli di dalam komputer anda. Sebelum itu, pastikan komputer anda telah dipasang dengan node.js. Sila rujuk hantaran saya mengenai pemasangan node.js menggunakan nvm terlebih dahulu jika belum memasang node.js. Untuk tutorial ini, saya menggunakan Mac. OK, tidak perlu berceloteh panjang, sila ikuti langkah-langkah di bawah.

Langkah #1

Pastikan anda telah memasang node. Periksa versi node anda.

Langkah #2

Jika node telah dipasang, jalankan perintah di bawah.

Arahan di atas akan memasang angular cli versi terbaharu. Sila tunggu sehingga pemasangan selesai. Setelah selesai, jalankan perintah di bawah.

Yeay, anda telah berjaya membuat pemasangan Angular cli ke dalam komputer anda.

Langkah #3

Buka terminal anda, pergi ke direktori projek anda dan jalankan perintah ini.

Arahan di atas adalah untuk menjana projek baharu dengan nama my-first-angular. Tunggu sehingga muat turun dan pemasangan selesai.

Langkah #4

Masuk ke direktori my-first-angular menerusi terminal dan jalankan perintah di bawah untuk kompil projek.

Langkah #5

Buka pelayar web anda, dan taipkan url http://localhost:4200

Ya, anda telah berjaya memasang angular cli di dalam komputer anda dan telah membuat projek baharu menggunakan angular 8 (versi semasa penulisan ini dibuat). Selamat berjaya.

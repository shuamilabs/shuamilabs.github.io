Pemasangan Homebrew di dalam macOS

posted on 8 Jun 2020 by Ahmad Shuami

Assalamualaikum dan Salam Sejahtera.

Perkongsian hari ini adalah berkenaan langkah-langkah untuk memasang hombrew di dalam komputer Mac. Homebrew merupakan Package Manager untuk memasang perisian di dalam Mac anda. Jom kita lihat cara-cara menggunakan perintah di command prompt. Caranya mudah dan cepat. sila ikuti langkah-langkah di bawah.

Langkah #1

Pastikan anda telah memasang xcode command line di dalam komputer Mac anda.

Langkah #2

Jika anda telah memasang xcode command line, teruskan dengan pemasangan Homebrew seperti di bawah.

Langkah #3

Tunggu sehingga pemasangan selesai. Anda perlu memasukkan kata laluan semasa pemasangan ini dibuat. Setelah selesai, anda boleh mengesahkan versi homebrew yang telah dipasang.

Langkah #4

Untuk memastikan pemasangan telah benar-benar berjaya dan tiada sebarang masalah atau konflik, jalankan perintah di bawah. Brew akan senaraikan langkah yang perlu dibuat jika berlaku sebarang masalah atau konflik.

Langkah #5

Ok, anda sudah berjaya memasang Homebrew di dalam komputer anda. Anda boleh membuat pemasangan sebarang perisian menggunakan homebrew antaranya PHP, MariaDB, Nginx dan lain-lain. Selamat mencuba.
